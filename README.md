Este proyecto implementa un sistema de ventas y facturación diseñado para una PYME, utilizando Programación Orientada a Objetos, principios de encapsulamiento, modularidad, abstracción, herencia, interfaces, y relaciones entre objetos.
Incluye repositorios, servicios, manejo de inventario, cálculos de impuestos, manejo de facturas y reportes.
